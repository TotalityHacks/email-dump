Create a .env file with the following (in KEY=VALUE format, one item per line):
1. DATABASE_URL: the URI of the database
2. MAILCHIMP_API_KEY: a MailChimp API key generated from https://admin.mailchimp.com/account/api/
3. MAILCHIMP_DC: the mailchimp datacenter (usually at the end of the API key; 'us15' is one example)
4. MAILCHIMP_LIST_ID: the ID of the list from the MailChimp web interface
